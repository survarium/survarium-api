#Clans

##Clans list
/clans - public list
/clans/cw - cw list

##Clan view
/clans/:abbr – totals public
/clans/:abbr/members – clan members
/clans/:abbr/matches – public matches stats
/clans/:abbr/cw – totals cw
/clans/:abbr/cw/matches – clan wars
