# Survarium-api

Statistics server based on API of [Survarium game](http://survarium.com)

## Docs
[API handlers](http://vaseker.github.io/survarium-api/docs/global.html)

## Setup
* `git clone https://github.com/vaseker/survarium-api.git!`
* `cd survarium-api`
* `npm run deps`

## Start
`npm start`
